### hot place 정보 조회

### home 화면에서 hot place 보여 줄때 사용

| 메소드 | 경로       | 설명                                     |
| ------ | ---------- | ---------------------------------------- |
| GET    | /hot_place | 홈화면에서 핫플레이스 리스트를 조회한다. |

#### 요청 헤더

```
Content-Type: application/json
Authoirzation: token
```

#### 응답 바디

**hot place  조회 성공시**

```json
{
    "status" : 200,
    "message" : "핫플레이스 리스트 조회 성공",
    "data" : {
        "contentIdx": 3,
        "userIdx": 1,
        "body": "컨텐츠 내용",
        "createDate": null,
        "likeCount": 5,
        "photo": [
           {
                "photoIdx": 9,
                "contentIdx": 3,
                "photoUrl": "사진 주소"
           }
        ],
        "auth": false,
        "like": false
    }
}
```

**조회 실패**

```json
{
```
