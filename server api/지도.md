## 지도

| 메소드 | 경로             | 설명                   |
| ------ | ---------------- | ---------------------- |
| POST   | /cafe/nearbycafe | 반경 3KM 주변카페 조회 |

#### 요청헤더

```
Content-Type: application/json
```

#### 요청 바디

```
{
	"cafe_latitude": "37.55048",
	"cafe_longitude": "126.9036",
	"cafe_id" : 5,  //카페 아이디
	"is_cafe_detail" : 0 // 카페 DetailView 밑에 주변 카페조회 하려면 값 1 / 지도탭에서 사용할 경우 값 0
}
```
